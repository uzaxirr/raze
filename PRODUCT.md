# Raze — Product Overview

> **One-liner:** Your crypto trading desk, inside a Telegram chat.

## The Problem

Crypto is powerful but the UX is broken. To make one informed trade on Solana, a user today has to:

1. **Check price** on Birdeye or CoinGecko
2. **Read sentiment** on Twitter/Reddit/LunarCrush
3. **Look up the token address** and verify it's not a scam
4. **Open a DEX** like Jupiter, connect a wallet, set slippage
5. **Monitor the transaction** on a block explorer
6. **Set a price alert** on yet another app to know when to exit

That's 5+ tools, 5+ tabs, and a ton of context switching — just for one trade. New users bounce. Experienced users waste time. Everyone loses alpha to friction.

## The Solution

Raze collapses the entire crypto workflow into a **single Telegram conversation**.

> "What's the sentiment on SOL?" → LunarCrush data
> "Swap 2 SOL for BONK" → Jupiter swap executed
> "Alert me if BONK drops 20%" → Price alert set
> "Send 5 SOL to vitalik.sol" → SNS resolved + transfer done

One chat. One agent. No tab-switching, no wallet UIs, no copy-pasting token addresses.

## How It Works

- **AI agent** (Grok) understands intent from natural language
- **7 MCP servers** handle specialized tasks (trading, research, alerts, etc.)
- **Embedded wallets** (Privy) remove key management friction
- **BYOMCP** lets power users plug in their own tools

**In short:** Raze turns Telegram into a full-stack Solana trading terminal — where the interface is just conversation.

- **Simplicity angle:** Text it like a friend, trade it like a pro.
- **AI angle:** An AI agent that turns your Telegram into a Solana trading terminal.
